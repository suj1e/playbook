# 力扣刷题手册

> 从零开始，系统学习力刷题，Java + Python 双语言实现

---

## 使用指南

本手册按照**数据结构**和**算法**两大维度组织内容，每个专题包含：

1. **README.md**：专题概述、学习路径、进度追踪
2. **templates.md**：代码模板，可直接复制使用
3. **题目文件**：每道题的详细解答，包含思路、代码、关键点

## 学习路线

### 第一阶段：基础（必做）

| 专题 | 预计时间 | 重点 |

| 03-linked-list | 2 天 | 反转、快慢指针、环检测 |

### 第三阶段：算法思想

| 专题 | 预计时间 | 重点 |

| 09-dp | 5 天 | 线性DP、背包、LCS |

| 11-greedy | 2 天 | 区间、调度 |

### 第四阶段：面试冲刺

| 专题 | 预计时间 | 重点 |

| 12-interview | 持续 | 高频题、公司真题 |

## 使用建议

### 对于新手

1. 先按顺序完成第一阶段的 4 个专题
2. 每道题先自己思考，再看解答
3. 理解思路后，自己动手实现
4. 定期复习，巩固知识

### 对于有基础者

1. 可以跳过熟悉的专题
2. 专注于薄弱环节
3. 多刷 medium 和 hard 题目
4. 总结自己的解题模板
