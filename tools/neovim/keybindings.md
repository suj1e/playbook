# Neovim/LazyVim 快捷键速查表

> 最常用的快捷键，提高编辑效率

---

## 1. 基础移动

| 快捷键 | 功能 |
|--------|------|
| `h j k l` | 左 下 上 右 |
| `w` / `b` | 下一个/上一个单词 |
| `e` / `ge` | 单词结尾/上一个单词结尾 |
| `0` / `$` | 行首/行尾 |
| `^` | 行首第一个非空字符 |
| `gg` / `G` | 文件首/文件尾 |
| `:n` | 跳到第 n 行 |
| `<Ctrl-d>` | 向下翻半屏 |
| `<Ctrl-u>` | 向上翻半屏 |
| `<Ctrl-f>` | 向下翻一屏 |
| `<Ctrl-b>` | 向上翻一屏 |

## 6. 搜索替换

### 搜索

| 快捷键 | 功能 |
|--------|------|
| `/pattern` | 向下搜索 |
| `?pattern` | 向上搜索 |

### 替换

| 命令 | 功能 |
|------|------|
| `:s/old/new/` | 替换当前行第一个 |
| `:s/old/new/g` | 替换当前行所有 |
| `:%s/old/new/g` | 替换文件所有 |
| `:%s/old/new/gc` | 替换所有（需确认） |
| `:%s/old/new/gi` | 替换所有（忽略大小写） |

## 10. 快捷键记忆技巧

### 移动类

- **hjkl** = 左下上右（键盘位置）
- **w** = word（单词）
- **b** = back（回退）
- **e** = end（结尾）
- **gg/G** = go/go（首尾）

### 编辑类

- **i** = insert（插入）
- **a** = append（追加）
- **c** = change（修改）
- **d** = delete（删除）
- **y** = yank（复制）
- **p** = paste（粘贴）

### 对象类

- **i** = inner（内部）
- **a** = around（周围）

### 模式类

- **v** = visual（可视）
- **V** = Visual line（行可视）
- **Ctrl-v** = Visual block（块可视）
